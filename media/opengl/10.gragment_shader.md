# Fragment Shader

```c
#version 300 es                            
precision mediump float;                   
out vec4 fragColor;                        
void main()                                
{                                          
   fragColor = vec4 ( 1.0, 0.0, 0.0, 1.0 );
}       
```

- precision 定义着色器浮点变量的默认精度。
